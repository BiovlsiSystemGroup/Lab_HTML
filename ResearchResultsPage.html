生醫與運動分析設計

本實驗主要為跨域系統應用開發，主旨在於讓學生在開發各種應用系統中培養自身的技術能力。應用方向主要為

智慧醫療、智慧工廠、健康促進、車用電子、元宇宙、運動科技。


主要研發技術在『整合系統設計』中的各部分。包含

  1. 電路設計
  2. 韌體設計
  3. 軟體設計(包含APP與電腦端軟體)
  4. 3D建模與3D列印
  5. AI應用系統
  6. 雲端系統

實驗室資源，包括：雲端AI伺服器、FPGA開發板、Microchip開發板、Cypress開發板、Labview系統、AR/VR系統、 3D列印機、電路設計所需之各種量測設備。

跟著社會與產業所需以及科技演進技術來進行系統發展，在各領域皆有多方面的創意應用實現。由於歷年開發系統過多，且多為產學合作無法列上，研究成果中僅摘錄部分應用例子。

上肢復健訓練

  1. 中風患者手部訓練用之即時分離混合肌電訊號系統
  2. 具上肢肌電訊號回饋控制之功能性電刺激系統
  3. 即時分離混合肌電訊號之回饋控制功能性電刺激系統

下肢復健訓練

  1. EMG回饋控制之功能性電刺激踩車系統
  2. 肌電訊號應用在即時回饋控制功能性電刺激踩車系統
  3. 雙向肌肉電刺激晶片與踏力感測踩車研製
  4. 家用功能性電刺激踩車
  5. 復健用遊戲式簡易踩車系統設計

居家應用

  1. 實現具有無線ANT網路跳點技術之居家照護系統
  2. 三軸加速度計應用於居家照護即時跌倒偵測系統

3D影像與視覺

  1. 即時動態立體影像顯示系統
  2. 立體影片改善眼睛疲勞之影響
  3. 以FPGA實現即時影像放大
  4. 立體影片參數變化對觀看者生理訊號影響之研究

一般民眾：低頭族、運動訓練

  1. 分析腹直肌在核心訓練下肌肉疲勞之情形
  2. 肢體不平衡量測與運動訓練系統
  3. 攜帶型即時生醫訊號量測與記錄之軟硬體整合系統
  4. 以疲勞與生理訊號關聯性建立即時疲勞評估系統。
  5. 紡織型足底壓力量測系統之研發

生醫IC

  1. 雙通道多頻率之疼痛舒緩電刺激系統設計
  2. 參數可調之疼痛舒緩電刺激晶片設計與實現
  3. 應用於異常心跳偵測之頻率電壓轉換電路
  4. 使用高壓製程之50mA雙相脈波電刺激晶片
  5. 可調控式雙相脈波電刺激晶片
